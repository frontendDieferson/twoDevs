# Passos a seguir


#### Fazendo o download do código fonte
```
# git clone https://github.com/frontendDieferson/twoDevs.git
```

#### Executando o aplicativo
```
# cd twoDevs
# npm install
# npm start
```

### Executando no navegador

Abra no seu navegador o endereço http://127.0.0.1:3000
